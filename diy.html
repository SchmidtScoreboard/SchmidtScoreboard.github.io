Build Your Own

I am working hard on a true guide to set up a Scoreboard from start to finish. I'll include a full parts list, API key generator, and everything else you might need.

Until then, here are links to all the source code with some basic setup documention:

  • Scoreboard: This repo contains the Python program that fetches data, displays scores, and handles setup
  • Controller App: This repo contains the Flutter cross-platform app to manage setup/change modes
  • AWS-Graph: This repo contains the API that runs on AWS that fetches data from other sources and consolidates it

I would greatly appreciate contributions to any or all of these projects. Got an idea for a new screen? Make a fork and send me a pull request! Or, shoot me an email at schmidtwmark+scoreboard@gmail.com
